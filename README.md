# Glassmorphism Generator

Generate the perfect glassmorphism CSS with a live preview — blur, transparency, border, and shadow in one visual editor.

## Features

- Adjustable blur radius, background opacity, and saturation
- Border opacity and color control
- Box shadow editor
- Background scene preview (gradient or image upload)
- One-click CSS copy
- Dark and light mode output
